Curelex – Smart Healthcare Web Application

📌 Overview
Curelex is a responsive healthcare web application built using HTML, CSS, and JavaScript. The platform includes role-based authentication for Patients and Doctors along with a structured dashboard interface.

The project focuses on clean UI design, responsive layouts, and scalable frontend architecture while simulating a real-world healthcare system workflow.

---

 🚀 Features

🏠 Homepage
- Modern and responsive landing page
- Clean navigation structure
- Informative healthcare sections

 👤 Patient Authentication
- Patient Sign Up
- Secure Login Interface
- User-friendly form validation

 🩺 Doctor Authentication
- Doctor Registration Form
- Professional details collection (Specialization, Registration Number, Hospital, etc.)
- Approval-based login system (no direct sign-in before approval)

 📊 Dashboard
- Structured dashboard layout
- Role-based interface design
- Organized UI components for future scalability
